# E-Commerce-Sales-Analysis-MySQL


Project Objective
To analyze an e-commerce dataset using SQL to identify top customers, understand repeat purchase behavior, and analyze revenue trends for business insights.

Dataset Overview
- Transaction ID
- Sale date & time
- Customer demographics (age, gender)
- Product category
- Quantity, price, COGS
- Total sales revenue

Key Analysis Performed
- Identified top customers based on total spending
- Analyzed repeat orders to measure customer loyalty
- Studied monthly revenue trends to understand business growth
- Performed profit analysis using COGS
- Segmented customers by age group and gender

Skills Used
- MySQL
- Aggregations (SUM, AVG, COUNT)
- GROUP BY, HAVING
- CASE WHEN
- Date functions
- Data cleaning
